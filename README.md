# C Practice
Just me doing my C Programming and Github practice in this repo.

---

### How to Run
Install GCC Compiler from [here](https://sourceforge.net/projects/gcc-win64/) and to verify run:
```bash
gcc --version
```
Save the file with `.c` extension and:
```bash
gcc filename.c -o output
./output
```
